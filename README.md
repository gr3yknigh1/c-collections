# cboxes

Simple collections library written in C

## Build

**Requirements**:

- GNU Make
- LLVM Clang
- (optional) [Criterion](https://github.com/Snaipe/Criterion) (for unittesting)

```bash
make all
```

## Run tests

Be sure to install Criterion unittesting library and run this command:

```bash
make clean all tests
```
